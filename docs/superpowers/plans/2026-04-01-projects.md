# Projects Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Linear-style project management app with Kanban board, list view, drag-and-drop, task detail panel, and project sidebar within the 500Claw workspace.

**Architecture:** Projects stored in Supabase Postgres with RLS via existing security definer functions. Kanban drag-and-drop via HTML5 API with optimistic UI updates. Task detail in a slide-over panel. Service client for atomic task counter increment. Same patterns as Messages sub-project.

**Tech Stack:** Next.js 16, Supabase Postgres, TypeScript, Tailwind CSS, Lucide icons, HTML5 Drag and Drop API

**Important:** Next.js 16 — `params` is a Promise. Use `createServiceClient()` for task creation (atomic counter). Use service client in layout for member/profile resolution (FK join workaround). No test runner — verify via browser.
